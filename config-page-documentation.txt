配置管理页面文档 (src/app/config/page.tsx)
=============================================

## 页面概述
系统配置管理页面，支持多种类型的配置项管理，包括文本、图片、富文本等内容的动态配置。

## 主要功能
1. **配置列表管理**
   - 显示所有系统配置项
   - 支持启用/禁用状态切换
   - 排序功能

2. **配置类型支持**
   - TEXT: 单行文本
   - TEXTAREA: 多行文本
   - IMAGE: 单图上传
   - MULTI_IMAGE: 多图上传
   - MULTI_TEXT: 多文本项
   - MULTI_CONTENT: 多文本图片组合
   - RICH_TEXT: 富文本编辑

3. **CRUD 操作**
   - 新增配置项
   - 编辑配置内容
   - 删除配置项
   - 状态管理

## 技术特点
- **富文本编辑**: 集成 WangEditor 组件
- **文件上传**: 支持图片上传和预览
- **表单验证**: 键名格式验证，必填项检查
- **响应式设计**: 适配不同设备尺寸
- **动态表单**: 根据配置类型动态渲染表单项

## 核心组件
- AdminLayout: 管理后台布局
- Table: 配置列表展示
- Modal: 编辑/新增弹窗
- Form: 动态表单
- Upload: 图片上传组件
- WangEditor: 富文本编辑器

## 数据结构
```
Config {
  id: number
  title: string        // 配置标题
  key: string          // 配置键名
  type: ConfigType     // 配置类型
  description: string  // 描述信息
  sort: number         // 排序值
  isEnabled: boolean   // 启用状态
  value: any           // 配置值
}
```

## API 接口
- GET /api/configs - 获取配置列表
- POST /api/configs - 新增配置
- PUT /api/configs/[id] - 更新配置
- DELETE /api/configs/[id] - 删除配置
- PATCH /api/configs/[id]/status - 状态切换

## 文件上传
- 上传路径: /api/upload/media
- 支持格式: 图片文件
- 鉴权方式: Bearer Token
- 文件大小限制: 根据配置类型

## 使用场景
- 网站基础信息配置
- 轮播图管理
- 公告内容管理
- 联系方式配置
- 多媒体资源管理

## 注意事项
1. 键名添加后不可修改，用于程序调用
2. 富文本内容支持图片和视频
3. 多图上传最多支持10张
4. 配置项按排序值升序显示
5. 禁用状态的配置不会在前台显示
